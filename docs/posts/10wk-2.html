A. lambda

- 예시1: \(x \to x+2\)

f = lambda x: x+2

- 예시2: \(x,y \to x+y\)

f = lambda x,y: x+y

- 예시3: ‘2023-09’ \(\to\) 9

f = lambda x: int(x[-2:])
f('2023-09')
9

- 예시4: ‘2023-09’ \(\to\) (2023,9)

f = lambda x: (int(x[:4]),int(x[-2:]))
f('2023-09')
(2023, 9)

- 예시5: 문자열이 ‘cat’이면 1 ’dog’ 이면 0 // ’cat이면 1 ’cat’이 아니면 0

f = lambda x: 1 if x=='cat' else 0
